## DATABASE IMPORT GUIDE

To import your inventory:

```bash
# 1. Start PostgreSQL
cd unistyles-website
docker-compose up -d postgres

# 2. Initialize database
docker exec -i unistyles-postgres psql -U unistyles -d unistyles < database/init.sql

# 3. Import products (script needed)
node scripts/import-products.js
```
